# 내 AI 워크스페이스

AI와 함께 일하는 공간입니다. 아래 폴더 구조만 알면 됩니다.

## 폴더 안내

| 폴더 | 용도 | 직접 관리? |
|------|------|-----------|
| `projects/` | 진행중인 작업. 폴더 하나가 프로젝트 하나 | AI가 자동 생성 |
| `notes/` | 메모, 아이디어, 생각 정리 | 자유롭게 사용 |
| `sessions/` | AI 대화 기록 (자동 저장) | 건드릴 필요 없음 |
| `archive/` | 끝난 프로젝트 보관소 | AI가 이동 제안 |

## 사용법

1. VSCode에서 이 폴더를 엽니다
2. Claude Code 터미널에서 하고 싶은 일을 말합니다
3. AI가 알아서 파일을 만들고, 정리하고, 저장합니다

## 예시

- "지난 달 마케팅 데이터 분석해줘" → `projects/마케팅-데이터-분석/` 자동 생성
- "오늘 회의 내용 정리해줘" → `notes/daily/` 에 저장
- "지난 주에 뭐 했는지 알려줘" → `sessions/` 에서 확인

## 자동 저장

모든 작업은 자동으로 저장됩니다. 별도로 저장 버튼을 누를 필요 없습니다.

## 도움이 필요하면

Claude Code에 "도와줘" 또는 "사용법 알려줘"라고 말하면 됩니다.
