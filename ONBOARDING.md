# 처음 5분 가이드

환영합니다! AI 워크스페이스가 준비되었습니다. 아래 순서대로 따라해보세요.

---

## 1단계: VSCode 열기

터미널(맥: Terminal, 윈도우: PowerShell)에서 아래 명령어를 입력하세요:

```
code ~/workspace
```

또는 VSCode를 직접 열고 → 파일 → 폴더 열기 → `workspace` 폴더 선택

---

## 2단계: Claude Code 시작하기

VSCode 하단의 터미널 패널에서 아래를 입력하세요:

```
claude
```

처음 실행하면 로그인이 필요합니다. 화면 안내를 따라 로그인하세요.

---

## 3단계: 첫 대화 해보기

로그인 후 Claude와 바로 대화할 수 있습니다. 아래 예시를 그대로 입력해보세요:

**예시 1 — 간단한 요청**
```
오늘 할 일 목록을 만들어줘
```
→ `notes/daily/` 폴더에 오늘 날짜로 파일이 자동 생성됩니다.

**예시 2 — 파일 분석**
```
이 CSV 파일 분석해줘
```
→ 분석하고 싶은 파일을 워크스페이스에 넣고 요청하세요.

**예시 3 — 아이디어 정리**
```
새 프로젝트 아이디어가 있어. 온라인 마케팅 캠페인을 기획하려고 해.
```
→ `projects/온라인-마케팅-캠페인/` 폴더가 자동 생성됩니다.

---

## 4단계: 기본 사용법

### 파일 관리는 AI가 합니다
- 폴더를 직접 만들 필요 없습니다
- 작업을 시작하면 AI가 알아서 프로젝트 폴더를 만들고 파일을 정리합니다
- 모든 작업은 자동 저장됩니다

### 말로 시키면 됩니다
- "지난 주에 뭐 했는지 알려줘" → 세션 기록에서 찾아줍니다
- "이 프로젝트 끝났어" → 완료 처리하고 보관함으로 옮깁니다
- "도와줘" → 사용법을 안내합니다

### 대화를 끝내려면
터미널에 아래를 입력하세요:
```
/exit
```
대화 내용은 자동으로 요약되어 `sessions/` 폴더에 저장됩니다.

## 5단계: 자주 쓰는 표현

| 이렇게 말하면 | 이렇게 됩니다 |
|-------------|-------------|
| "분석해줘" | 데이터를 분석하고 인사이트를 정리 |
| "정리해줘" | 내용을 구조화해서 문서로 저장 |
| "요약해줘" | 긴 내용을 핵심만 정리 |
| "초안 만들어줘" | 문서/메일/보고서 초안 작성 |
| "비교해줘" | 여러 옵션을 비교 분석 |
| "템플릿 만들어줘" | 반복 업무용 양식 생성 |
