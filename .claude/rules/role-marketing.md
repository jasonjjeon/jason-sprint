# 마케팅 역할 규칙

## 나는 누구인가
이 워크스페이스의 사용자는 마케팅 담당자입니다.
광고 채널 운영, 소재 기획/분석, 미디어 믹스, 대행사 관리가 주 업무입니다.

## 업무 맥락
- 주요 광고 채널: 구글, 네이버, 카카오, 크리테오, 틱톡, 메타
- 에어브릿지 등 어트리뷰션 툴로 성과 추적
- 대행사(카페24 등)와 협업하여 소재 제작 및 프로모션 혜택 관리
- 주간 소재 분석 미팅 진행

## AI 지시사항

### 데이터 분석 시
- 광고 성과 데이터를 분석할 때는 항상 CTR, CPC, ROAS, CPA 기준으로 정리
- 채널별 비교 시 표 형태로 한눈에 보이게
- 인사이트는 "다음 액션"까지 제안 (예: "크리테오 예산을 네이버로 10% 이동 검토")

### 소재 기획 시
- 타겟 성별/연령을 먼저 확인
- 경쟁사 레퍼런스가 있으면 참고
- 카피 제안 시 A/B 테스트용 2-3개 버전 제공

### 보고서 작성 시
- 주간/월간 보고서는 "요약 → 채널별 상세 → 인사이트 → 다음 주 계획" 구조
- 숫자는 전주 대비 증감(%)으로 표현
- 미디어 믹스 변경 제안은 근거 데이터와 함께

### 파일 정리
- 광고 데이터 파일: `projects/프로젝트명/data/` 하위에 저장
- 소재 기획서: `projects/프로젝트명/소재/` 하위에 저장
- 보고서: `projects/프로젝트명/보고서/YYYY-MM-DD_제목.md`
