# 자주 묻는 질문 (FAQ)

---

## 기본

### Q: 터미널이 뭔가요?
컴퓨터에 글자로 명령을 내리는 창입니다.
- **맥**: Spotlight(Cmd+Space)에서 "Terminal" 검색
- **윈도우**: 시작 메뉴에서 "PowerShell" 검색

### Q: VSCode가 뭔가요?
코드 편집기인데, 여기서는 AI와 대화하는 도구로 씁니다. 메모장의 업그레이드 버전이라고 생각하면 됩니다.


---

## 설치 문제

### Q: `npx bind-ai-setup` 입력했는데 에러가 나요
**"command not found" / "명령을 찾을 수 없습니다"**
→ Node.js가 설치되지 않았습니다.
- 맥: https://nodejs.org/ko/download 에서 다운로드
- 윈도우: https://nodejs.org/ko/download 에서 다운로드
- 설치 후 터미널을 닫았다가 다시 열고 재시도

**"EACCES" / "permission denied"**
→ 권한 문제입니다.
- 맥: `sudo npx bind-ai-setup`으로 재시도
- 윈도우: PowerShell을 "관리자 권한으로 실행"

**"ENOTFOUND" / "network error"**
→ 인터넷 연결을 확인하세요.

### Q: Git이 없다고 나와요
- 맥: 터미널에 `xcode-select --install` 입력
- 윈도우: https://git-scm.com/download/win 에서 다운로드

## 사용 중 문제

### Q: 빨간색 에러 메시지가 나와요
대부분 심각한 문제가 아닙니다. 에러 메시지를 그대로 Claude에게 보여주면 해결 방법을 알려줍니다:
```
이런 에러가 나왔어: [에러 메시지 붙여넣기]
```

### Q: "API 키"가 뭔가요? 입력하라고 하는데요
AI 서비스 이용권 같은 것입니다. 관리자에게 문의하세요. 직접 만들 필요 없습니다.

### Q: 파일이 어디에 저장됐는지 모르겠어요
Claude에게 물어보세요:
```
방금 만든 파일 어디 있어?
```
또는 VSCode 왼쪽 탐색기 패널에서 `projects/` 폴더를 확인하세요.

### Q: 실수로 파일을 지웠어요
자동 저장(Git) 덕분에 복구할 수 있습니다. Claude에게:
```
방금 지운 파일 복구해줘
```

### Q: 예전에 했던 작업을 다시 보고 싶어요
```
지난 주에 뭐 했는지 알려줘
```
`sessions/` 폴더에 날짜별 대화 요약이 저장되어 있습니다.

## 보안

### Q: 회사 데이터를 입력해도 되나요?
관리자가 정한 보안 정책을 따라주세요. 일반적으로:
- **OK**: 공개 데이터, 일반 업무 내용
- **주의**: 내부 문서 (관리자 확인 필요)
- **금지**: 비밀번호, API 키, 고객 개인정보 (이름/연락처/주민번호 등)
